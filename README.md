# SistemaReservaHotel
Este proyecto es como parte de la Ingeniería de Software Orientada a Servicios (SOA) en la maestría de la Universidad CESUMA. Sistema de gestión de reservas para cadena hotelera implementado con Arquitectura Orientada a Servicios (SOA), desarrollado como caso práctico académico para la asignatura de Ingeniería de Software Orientada a Servicios.
